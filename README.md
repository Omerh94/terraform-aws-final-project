# Terraform AWS Infrastructure Project

## Overview

This project demonstrates the deployment of a production-style AWS infrastructure using Terraform.

The infrastructure was divided into two separate parts:

- **Part 1** - Network Infrastructure
- **Part 2** - Bastion Host Infrastructure

The project includes:

- AWS VPC
- Public and Private Subnets
- DB Subnets
- Internet Gateway
- NAT Gateways
- Route Tables
- Security Groups
- Bastion Host
- Launch Template
- Auto Scaling Group
- Remote State stored in Amazon S3

The project was developed as part of the DevOps course at Real Time College.
